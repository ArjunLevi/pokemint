**PokeMint** is a decentralized application built on the **Base Network** that allows users to catch and mint unique Pokemon-themed NFTs. This project leverages the Farcaster ecosystem, functioning as a high-performance Mini App for the next generation of onchain gamers.

⚙️ **Stack:** NextJS, RainbowKit, Hardhat, Wagmi, Viem, and Typescript. Optimized for **Base Sepolia**.

- ✅ **Farcaster Native**: Built-in support for Farcaster Mini App SDK and Frame metadata.
- 🪝 **Onchain Minting**: Seamless ERC-721 minting directly to your wallet.
- 🧱 **Base Ecosystem**: Optimized for low-cost, lightning-fast transactions on the Base Network.
- 🔥 **Mini App Integration**: Launchable directly from Warpcast with custom action buttons.

## 🚩 Project: 🎟 PokeMint Tokenization

🎫 **Catch, Mint, and Battle on Base.** PokeMint turns your favorite monsters into unique digital collectibles. This project explores the power of onchain ownership, where every "catch" is an immutable transaction and every "monster" is a unique asset you truly own.

🌟 **The Goal:** A seamless minting experience that feels like a game but functions as a robust NFT ecosystem. Deploy your contracts, catch your first PokeMint, and share your collection across the Farcaster social graph!
